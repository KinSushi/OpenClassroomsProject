# 🏖️ Reservia — Site vitrine agence de voyage

Intégration d'une maquette de site vitrine pour l'agence de voyage fictive **Reservia**, réalisée dans le cadre du **Parcours Développeur WordPress d'OpenClassrooms (Projet 3)**.
Le projet respecte les contraintes suivantes : HTML5 sémantique, CSS pur avec variables personnalisées, approche mobile-first, sans aucun framework.

## ✨ Fonctionnalités

- **Navigation responsive** avec menu hamburger (mobile) et barre de navigation classique (desktop)
- **Formulaire de recherche** intégré dans le header avec champ texte et bouton de validation
- **Filtres de recherche** par catégories (hôtels, appartements, auberges…) avec effets hover
- **Cartes hébergements** affichées en grille responsive avec photo, titre, note et prix
- **Section activités** en mosaïque avec images pleine hauteur et légendes
- **Footer** multi-colonnes avec liens organisés par thématique
- Gestion complète des **états interactifs** : hover, focus, active sur tous les éléments cliquables

---

## 🛠️ Stack technique

| Technologie | Usage |
|---|---|
| **HTML5 sémantique** | `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>` |
| **CSS3** | Flexbox, Grid, positionnement, transitions |
| **Variables CSS** | Palette de couleurs et espacements centralisés dans `:root` |
| **Google Fonts** | Police Raleway |
| **Responsive mobile-first** | Breakpoints : `768px` (tablette) · `1024px` · `1200px` (desktop) |
| **JavaScript vanilla** | Burger menu (toggle classe active) |
